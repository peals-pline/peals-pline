# Hi, I'm Denis Dyakov

I'm a UX/UI designer working on thoughtful digital products, web interfaces, product flows, visual systems, and clean user experiences for companies, teams, and independent projects.

Alongside my professional design work, I build and maintain open-source tools focused on education, creative workflows, privacy, and productivity.

I care about interfaces that feel clear, calm, and useful — products that are simple to understand, pleasant to use, and built with attention to structure, details, and long-term maintainability.

## What I do

* UX/UI design for web and digital products
* Product flows, wireframes, prototypes, and interface concepts
* Visual design systems, components, and interface guidelines
* Landing pages, dashboards, SaaS interfaces, and product websites
* Design thinking, usability improvements, and product polish
* Local-first open-source tools with clear UX and practical use cases

## Professional focus

I design digital products that help users move through complex flows with less friction.

My work usually focuses on:

* clear information architecture
* simple and understandable user flows
* clean visual hierarchy
* reusable UI components
* responsive layouts
* accessible interface patterns
* thoughtful product presentation

## Maintained open-source projects

I maintain small, practical open-source tools that connect my design background with real product needs.

### TutorDesk

A local-first workspace for private tutors and small educators.

TutorDesk helps tutors track students, lessons, homework, recurring mistakes, progress, and reports without sending student data to third-party servers.

Focus areas:

* education workflows
* student progress tracking
* local-first data storage
* privacy-conscious reports
* accessibility and responsive UX

### DesignShelf

A local-first design library for UX/UI designers.

DesignShelf helps designers organize fonts, color palettes, design systems, UI references, copy snippets, inspiration, and project kits in a private searchable workspace.

Focus areas:

* UX/UI design workflows
* design systems and design tokens
* project-kit organization
* searchable creative references
* optional future AI/model-assisted tagging and brief generation with user control

### ReadLaterCleaner

A privacy-friendly browser extension for cleaning and organizing read-later links.

ReadLaterCleaner helps users save, review, tag, archive, deduplicate, and export links without analytics or a remote backend.

Focus areas:

* productivity
* browser workflows
* local storage
* backlog cleanup
* Markdown and JSON export

## Design principles

* Clarity before decoration
* Structure before visual polish
* Good UX should reduce effort, not add complexity
* Interfaces should feel calm, useful, and understandable
* Visual systems should help products scale
* Privacy-friendly tools should be useful without forcing accounts or cloud storage
